Tentang Sentra IoT

Sentra Internet of Things (IoT) di SMK Telkom Jakarta adalah program intrakurikuler yang dirancang untuk memberikan pengalaman praktis dalam merancang, merakit, dan memprogram perangkat berbasis mikrokontroler yang terhubung ke internet.

Materi Pembelajaran:

  • Pemrograman dasar untuk perangkat IoT
  • Pengelolaan data dari sensor
  • Perakitan dan konfigurasi perangkat keras
  • Penggunaan sensor dan aktuator
  • Integrasi sistem dengan jaringan internet

Tujuan Pembelajaran:

  • Memahami konsep dasar sistem kendali digital
  • Menerapkan prinsip kerja logika urutan dan pengendalian output
  • Mengintegrasikan perangkat keras sederhana sebagai fondasi pengembangan perangkat IoT yang lebih kompleks
  • Mempersiapkan siswa menghadapi perkembangan dunia digital yang pesat

Deskripsi Proyek

Proyek ini merancang sistem pemberi pakan ikan otomatis menggunakan Arduino Uno sebagai pengendali utama. Sistem memanfaatkan sensor ultrasonik HC-SR04 untuk mendeteksi keberadaan objek pada jarak tertentu (≤5 cm). Ketika objek terdeteksi, mikrokontroler memproses sinyal dan menggerakkan motor servo untuk membuka mekanisme penyaluran pakan dari wadah ke dalam akuarium. Seluruh proses berlangsung otomatis tanpa intervensi manual.

Tujuan Proyek

  • Mendokumentasikan proses perancangan dan implementasi sistem pemberi pakan ikan otomatis.
  • Memberikan pemahaman mengenai penerapan sensor jarak dan aktuator dalam sistem otomatisasi.
  • Melatih keterampilan siswa dalam merakit perangkat keras serta menulis program menggunakan Arduino IDE.

Komponen yang Digunakan

Arduino Uno
Sensor Ultrasonik HC-SR04
Motor Servo (2 buah)
Kabel Jumper
Breadboard
Aquarium 11×11 cm

Cara Kerja Sistem

  1. Sensor ultrasonik mendeteksi jarak objek di sekitar akuarium.
  2. Jika jarak objek ≤ 5 cm, sistem menganggap ada objek terdeteksi.
  3. Arduino memproses data dan mengirim sinyal ke motor servo.
  4. Servo bergerak membuka mekanisme wadah pakan → pakan jatuh ke akuarium.
  5. Setelah beberapa detik, servo kembali ke posisi awal, sistem siap mendeteksi ulang.

Kode Program Arduino

Berikut adalah kode program lengkap yang digunakan untuk mengendalikan sistem pemberi pakan ikan otomatis menggunakan Arduino Uno, sensor ultrasonik HC-SR04, dan motor servo.

pakanIkan.ino Arduino IDE (C++) 
#include <Servo.h>

const int trigPin = 9;
const int echoPin = 10;
const int servoPin = 6;

Servo myServo;

long duration;
int distance;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  myServo.attach(servoPin);
  myServo.write(90); // posisi awal servo (tengah)
  Serial.begin(9600);
}

int getDistance() {
  // Kirim pulsa trigger
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  // Baca durasi echo
  duration = pulseIn(echoPin, HIGH);

  // Hitung jarak dalam cm
  int dist = duration * 0.034 / 2;
  return dist;
}

void loop() {
  distance = getDistance();

  Serial.print("Jarak: ");
  Serial.print(distance);
  Serial.println(" cm");

  if (distance > 0 && distance <= 5) {
    Serial.println("Objek terdeteksi! Servo berputar...");

    // Putar servo searah jarum jam (0 derajat) selama 1 detik
    myServo.write(0);
    delay(1000);

    // Kembalikan ke posisi tengah
    myServo.write(90);
    delay(500);
  }

  delay(100);
}

Penjelasan:

  • trigPin (D9) & echoPin (D10) : pin untuk sensor ultrasonik.
  • servoPin (D6) : pin untuk mengontrol motor servo.
  • myServo.write(90) : servo di posisi tengah (palang tertutup).
  • getDistance() : fungsi menghitung jarak dalam cm.
  • Jika jarak ≤ 5 cm, servo berputar ke 0° (membuka) selama 1 detik, lalu kembali ke 90°.
